1. Тема та Постановка задачі

Тема: Структура HTML-документа. Вибір предметної області. Розробка структури веб-сайту.

Мета: Придбати практичні навички роботи з HTML-документом, створення структури проекту, використання основних тегів розмітки, таблиць, форм, посилань та зображень. Реалізувати багатосторінковий сайт.

Завдання: Розробити веб-сайт "Bookshelf - Магазин Книг" з каталогом, детальним описом книг та кошиком.

2. Посилання на проєкт

Репозиторій GitHub Живий сайт (Pages)

3. Структура проєкту

Проєкт складається з HTML сторінок, файлів стилів (CSS), скриптів (JS) та зображень.

Рис. 1. Файлова структура проєкту на GitHub
  • index.html - Головна сторінка зі слайдером.
  • page.html, page2.html, page3.html - Сторінки опису окремих книг.
  • st/ - Папка зі стилями (style.css, style2.css...).
  • im/ - Папка із зображеннями.
  • main.js - Скрипт для логіки слайдера та модальних вікон.
